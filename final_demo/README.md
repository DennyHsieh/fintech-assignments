# Fintech Final Project - Spring 2019
Group 10

## Setup

- Setup on Python 3.6.

```
pip install flask
pip install jinja2
```
## How to Run
### SQLite

- Create the database manually.

```bash
python setup_db.py
```

### Then, Run Server

```bash
python demo.py
```

- Will run a server listening <http://localhost:5000>.
- Enter the name you want to search.

- You can see the search result from the result page.
[](img/demo_result.png)
